Adding Screenshots

  1. Upload the image to your GitHub repo
    Go to your repo on GitHub → navigate to src/assets/ (create it if needed) → drag-and-drop your screenshot → commit.
  2. Copy the image URL
    Click the uploaded file → click Raw → copy the URL from your browser.
    It will look like:
    https://raw.githubusercontent.com/username/repo/main/src/assets/screenshot.png
  3. Insert it in your write-up
    Use standard Markdown image syntax:
    ![Description of image](paste-url-here)
